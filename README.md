# 3uttons | 按钮之谜 #

-------

Solutions

Usage:

0. Install NodeJS & NPM and run `npm install`
1. Type `npm start` in shell to run
2. Type the level you want to inspect
3. Got result
4. Operate -> 0 - Yellow button; 1 - Red button; 2 - Green button

Currently supports level 1 - 73
